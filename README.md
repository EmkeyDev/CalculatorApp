## 📌 Project Description

Smart Calculator is a desktop GUI application with a modern dark interface.
It supports two modes: Standard and Scientific, and automatically saves 
calculation history to a file.

---

## ❗ Problem Statement

Standard calculators are limited:
- No history of operations
- No scientific functions
- Outdated interface

This project solves all three problems in one application.

---

## ✅ Solution Overview

- Built with Python and CustomTkinter for a modern dark GUI
- Two modes: Standard and Scientific
- History automatically saved to a JSON file
- Clean, readable, and structured code split into 3 modules
